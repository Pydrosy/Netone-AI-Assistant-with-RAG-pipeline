# Netone AI Assistant

An intelligent customer service assistant for Netone Zimbabwe powered by RAG (Retrieval-Augmented Generation) technology.

## 🚀 Features

- **AI-Powered Chat**: Natural language conversations about Netone services
- **RAG Architecture**: Retrieves relevant information from company documents
- **Comprehensive Knowledge Base**: Includes bundles, pricing, company info, SIM cards, and more
- **Modern Frontend**: React-based chat interface with Tailwind CSS
- **FastAPI Backend**: High-performance Python async API
- **Groq Integration**: Fast LLM inference with llama-3.3-70b
- **FAISS Vector Store**: Efficient similarity search for document retrieval

## 📋 Knowledge Base Includes

- ✅ Company information and history
- ✅ Leadership and board members
- ✅ All bundle prices (BBB, MoGigs, OneFusion, etc.)
- ✅ SIM card information (including Tourist SIM)
- ✅ Customer support contacts
- ✅ USSD codes
- ✅ Technical support guides
- ✅ FAQs and troubleshooting

## 🛠️ Tech Stack

### Backend
- **Python 3.9+**
- **FastAPI** - Web framework
- **LangChain** - RAG orchestration
- **FAISS** - Vector similarity search
- **Groq** - LLM inference (llama-3.3-70b)
- **sentence-transformers** - Embeddings model

### Frontend
- **React 19** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **React Markdown** - Message formatting
