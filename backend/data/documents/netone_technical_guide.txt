# NETONE ZIMBABWE - TECHNICAL SUPPORT GUIDE

## COMMON NETWORK ISSUES AND SOLUTIONS

### No Network Signal
**Symptoms:** No bars, "No Service" on phone

**Troubleshooting Steps:**
1. **Restart your phone** - This refreshes the connection to the network
2. **Check coverage area** - Ensure you're in a Netone coverage zone
3. **Toggle Airplane Mode** - Turn on for 10 seconds, then off
4. **Manual network selection**:
   - Go to Settings → Mobile Networks → Network Operators
   - Select "Netone Zimbabwe" manually
5. **Check SIM card** - Remove and reinsert, clean contacts
6. **Update carrier settings** - iPhone: Settings → General → About
7. **Reset network settings** - Android: Settings → General Management → Reset → Reset Network Settings

**If issue persists:** Call 111 or visit nearest service center

### Slow Internet Speed
**Symptoms:** Web pages load slowly, videos buffer

**Troubleshooting Steps:**
1. **Check your data bundle** - Dial *123# to check remaining data
2. **Check signal strength** - Move to area with better coverage
3. **Close background apps** - Apps using data in background
4. **Clear browser cache** - Settings → Apps → Browser → Clear Cache
5. **Check APN settings**:
   - APN: netone.internet
   - Username: (leave blank)
   - Password: (leave blank)
   - Authentication type: PAP or CHAP
6. **Reset APN to default** - Delete and recreate APN
7. **Test at different times** - Peak hours (6-10 PM) may be slower

**Expected speeds:**
- 4G/LTE: 10-50 Mbps
- 3G: 2-8 Mbps
- 2G: 0.1-0.3 Mbps

### Can't Send/Receive SMS
**Symptoms:** Messages not sending, not receiving

**Troubleshooting:**
1. **Check message center number** - Should be +263 71 234 567
2. **Check if number is blocked** - Settings → Blocked numbers
3. **Clear messaging app cache** - Settings → Apps → Messages → Clear Cache
4. **Check SMS bundle** - Dial *123# to check remaining SMS
5. **Test with different recipient** - May be specific number issue

### Call Connection Issues
**Symptoms:** Calls drop, can't connect, poor audio quality

**Troubleshooting:**
1. **Check signal strength** - Low signal causes drops
2. **Disable WiFi Calling** if enabled and unstable
3. **Check if number is correct** - Dial carefully
4. **Try calling different number** - Isolate the issue
5. **Check call barring settings** - Settings → Call Settings → Call Barring

### Can't Access Internet After Bundle Purchase
**Symptoms:** Data not working even after buying bundle

**Solutions:**
1. **Restart your phone** - Refreshes network connection
2. **Toggle mobile data** - Turn off, wait 10 seconds, turn on
3. **Check if bundle activated** - Dial *123# to check balance
4. **Wait 10-15 minutes** - Sometimes activation takes time
5. **Remove and reinsert SIM** - During active data session
6. **Check if data roaming is on** (if outside home area)

## DEVICE-SPECIFIC SETTINGS

### Android APN Settings
**Path:** Settings → Connections → Mobile Networks → Access Point Names

**Create new APN:**
- Name: Netone Internet
- APN: netone.internet
- Proxy: Not set
- Port: Not set
- Username: Not set
- Password: Not set
- Server: Not set
- MMSC: Not set
- MMS Proxy: Not set
- MMS Port: Not set
- MCC: 648
- MNC: 04
- Authentication type: PAP or CHAP
- APN type: default,supl,mms
- APN protocol: IPv4/IPv6
- APN roaming protocol: IPv4
- Bearer: Unspecified

**Save and select this APN**

### iPhone APN Settings
**Note:** iOS may not allow manual APN configuration without carrier profile

**For newer iPhones:**
- APN settings are automatically configured from SIM card
- If needed, install carrier profile from Netone website

**For older iPhones:**
1. Go to Settings → Cellular → Cellular Data Network
2. Enter:
   - APN: netone.internet
   - Username: (blank)
   - Password: (blank)

### Windows Phone Settings
1. Go to Settings → Network & Wireless → Cellular & SIM
2. Select SIM settings
3. Add Internet APN:
   - APN: netone.internet
   - Username: (blank)
   - Password: (blank)
   - Type: IPv4

## ONEMONEY SERVICES

### OneMoney Registration
1. Dial *151# from your Netone line
2. Select "Register for OneMoney"
3. Enter your name as per ID
4. Create a 4-digit PIN
5. Confirm PIN
6. You'll receive confirmation SMS

### OneMoney Transactions

**Send Money:**
1. Dial *151*2#
2. Enter recipient's mobile number
3. Enter amount
4. Enter PIN
5. Confirm transaction

**Buy Airtime:**
1. Dial *151*3#
2. Select "My number" or "Other number"
3. Enter amount
4. Enter PIN
5. Confirm

**Pay Bills:**
1. Dial *151*4#
2. Select biller (ZESA, TelOne, etc.)
3. Enter account number
4. Enter amount
5. Enter PIN
6. Confirm

1. Dial *151*5#
2. Enter PIN
3. View balance

**Change PIN:**
1. Dial *151*6#
2. Enter old PIN
3. Enter new PIN
4. Confirm new PIN

## FREQUENTLY ASKED QUESTIONS

### General FAQs

**Q: How do I get my Netone number?**
A: Dial *130# and it will display your number

**Q: How do I check my phone's IMEI?**
A: Dial *#06#

**Q: How do I check network coverage in my area?**
A: Visit www.netone.co.zw/coverage or call 111

**Q: What should I do if my SIM card is lost/stolen?**
A: Visit nearest service center with ID for replacement

**Q: How long does SIM replacement take?**
A: Immediate, fee .00

### Data FAQs

**Q: Why is my data finishing quickly?**
A: Check background apps, video streaming, auto-updates

**Q: Does unused data roll over?**
A: Only on selected premium bundles

**Q: Can I share my data with others?**
A: Yes, via OneMoney or data gifting feature

**Q: How do I check data balance?**
A: Dial *124#

**Q: What is 4G+?**
A: Enhanced 4G with speeds up to 200 Mbps

### OneMoney FAQs

**Q: Is OneMoney free?**
A: Registration is free, some transactions have fees

**Q: How do I reset my PIN?**
A: Visit any Netone service center with ID

**Q: Can I send money to EcoCash?**
A: Yes, interoperability is available

**Q: What if I enter wrong PIN?**
A: After 3 wrong attempts, account is locked for 24 hours

**Q: Is my money safe?**
A: Yes, insured by Deposit Protection Corporation

### Roaming FAQs

**Q: Do I need to activate roaming before traveling?**
A: Yes, activate at least 24 hours before

**Q: Can I use my bundles while roaming?**
A: No, roaming has separate rates

**Q: How do I check roaming rates?**
A: Dial *126*2#

**Q: Will I receive SMS while roaming?**
A: Yes, standard SMS rates apply

**Q: Can I call customer care while roaming?**
A: Yes, +263 78 200 1111
